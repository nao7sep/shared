# Transition Guide

Guide for transitioning from Windows/C# to Mac/Python/TypeScript with AI-collaborative development.

## Reading Order

1. **01-mindset.md** - How to work with AI as a development partner
2. **02-project-setup.md** - Starting new FastAPI + React projects
3. **03-code-guidelines.md** - Balancing SOLID principles with Python idioms
4. **04-multi-project-workflow.md** - Managing 5+ projects simultaneously

## Usage

**First-time readers**: Read in order (01 → 04) for minimal redundancy and maximum clarity.

**Returning readers**: Jump to specific documents as needed:
- Feeling uncertain about delegation? → 01-mindset.md
- Starting a new project? → 02-project-setup.md
- Making architectural decisions? → 03-code-guidelines.md
- Setting up multi-project workflow? → 04-multi-project-workflow.md

## Context

These documents assume:
- Transitioning from C# to Python/TypeScript
- Transitioning from Windows to Mac
- Shifting from using AI as a writing helper to AI as a development partner
- Working on FastAPI backend + React/TypeScript frontend projects
- Managing multiple projects simultaneously
