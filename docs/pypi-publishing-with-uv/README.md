# PyPI Publishing with uv

Reference docs for publishing Python CLI apps to PyPI using **uv** as the build/publish tool.

## Audience

Solo developer. Assumes familiarity with Python, uv, and pyproject.toml.

## Reading Order

| # | File | What it covers |
|---|------|---------------|
| 1 | [01-setup.md](01-setup.md) | Accounts, API tokens, 2FA, token scoping |
| 2 | [02-project-structure.md](02-project-structure.md) | pyproject.toml, src layout, what gets packaged, dependencies |
| 3 | [03-publishing.md](03-publishing.md) | Versioning, build, publish to TestPyPI/PyPI, troubleshooting |
| 4 | [04-tooling.md](04-tooling.md) | `uv tool` commands, publish script automation |

Each doc assumes the previous ones were read.
